# Функциональные интерфейсы

_Функциональный интерфейс_ - это интерфейс, который определяет только один абстрактный метод.

Чтобы точно определить интерфейс как функциональный, добавлена аннотация @FunctionalInterface. Она обозначит замысел и не даст определить второй абстрактный метод в интерфейсе на этапе комплияции.
